# Spudmash Quick Hacks: Random User API: Web API 2.0

*Quick Hack Tasks:*

Integrate the <http://randomuser.me> REST service with:

- ASP.NET Web Api 2.0

Each tech will consist of:

- Responsive UI
- Service Layer
- Automated Test (UI, unit, integration)
